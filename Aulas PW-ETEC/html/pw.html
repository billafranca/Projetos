os tipos de css, 11/04/2025

inline interno externo
inline
O css inline é estilo do html em específico, ex:
<"body style="background-color: black; color: white;">
Externo
O css externo é o estilo fora do html, que conduz o html por outro arquivo (externamente) ex:style.css
Interno
O css interno, é a estilização do html, dentro do html, utilizando a tag style, ex: <"style><"/style">
